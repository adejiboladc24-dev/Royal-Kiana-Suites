# Royal Kiana Suites - Hotel Booking System

A full-stack hotel booking website with payment integration, email system, and staff management portal.

## 🌟 Features

- **26 Room Types** - From Standard to Presidential Suites
- **Payment Integration** - Bank Transfer, OPay, Pay at Hotel
- **Email System** - Password reset with 6-digit codes
- **Staff Portal** - Real-time booking management
- **Dark/Light Mode** - User preference theme toggle
- **Responsive Design** - Works on all devices
- **Scroll to Top** - Smooth navigation experience

## 🚀 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Framer Motion
- React Router

### Backend
- Node.js
- Express.js
- PostgreSQL
- Nodemailer
- JWT Authentication

## 📦 Installation

### Frontend Setup
```bash
cd royal-kiana-frontend
npm install
npm start
```

### Backend Setup
```bash
cd royal-kiana-backend
npm install
npm start
```

## 🔧 Environment Variables

### Backend (.env)
```
PORT=5000
DATABASE_URL=your_postgresql_url
JWT_SECRET=your_jwt_secret
EMAIL_USER=your_email@gmail.com
EMAIL_PASSWORD=your_app_password
```

### Frontend (.env)
```
REACT_APP_API_URL=http://localhost:5000
```

## 🌐 Deployment

### Vercel (Frontend)
The frontend is configured to deploy on Vercel automatically.

### Backend Deployment
Deploy backend separately on:
- Render
- Railway
- Heroku
